# Valkyra Context

Valkyra is a low-cost, serverless-first microSaaS for SMB and SME software teams that need security guardrails for AI-assisted coding.

## Product Direction

Build a GitHub-first control plane that helps teams using Codex, Cursor, Copilot, Claude Code, Windsurf, and similar agents install secure coding rules, scan pull requests, detect CI/CD risks, and produce security-readiness reports.

## Core Promise

AI coding is fast. Valkyra helps teams keep it safe enough for business.

## MVP

- GitHub App install and repository onboarding.
- Project CodeGuard-based policy profiles.
- Basic repository readiness scan.
- Pull request scan and comment workflow.
- CI/CD workflow risk checks for GitHub Actions.
- Secret detection using Gitleaks.
- SAST-style checks using Semgrep Community.
- Dependency checks using OSV Scanner or GitHub advisory data.
- Dashboard with risk score and recent findings.
- AI Coding Security Readiness report.
